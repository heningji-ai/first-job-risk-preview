# AGENTS.md

## 工作边界

1. 所有文件只能创建、修改在当前项目目录内。
2. 不得写入当前项目目录以外的任何位置。
3. 不得删除文件，除非我明确批准。
4. 不得自行接入第三方服务、API Key、数据库或微信 openid。
5. 不得自行编造完整题库、评分分值和风险卡业务内容。

## 开发流程

1. 每次执行前先说明计划。
2. 每次执行后说明改动了哪些文件。
3. 涉及 npm install、git push、删除文件、联网访问，必须先请求我确认。
4. 先阅读 `docs/PRODUCT_SPEC_V1.3.md`。
5. 第一阶段只允许输出文档，不允许开发完整业务代码。

## 项目优先级

1. 主流程跑通优先。
2. 配置化结构优先。
3. 评分和风险逻辑稳定优先。
4. 动画精致度、后台、支付、openid、保存图片都不是第一阶段目标。
