# RESULT_PAGE_CTA_CONVERSION_NOTES

## 1. 本阶段完成内容

ResultPage 底部私域承接区采用公众号关键词方式。

当前普通用户页只保留两个轻量入口：

- 公众号：猎头季哥人才重估实验室
- 企业微信：关注公众号后，回复【重估】获取添加方式

## 2. 当前 CTA 定位

CTA 的作用是让用户在看完路径适配风险预演后，知道后续可以继续看求职判断、简历、面试、Offer 和试用期相关内容。

本阶段不要求用户复制长话术，也不要求用户发送 JD、招聘截图或 offer 信息。

在路径适配风险预演页中，CTA 只做轻量承接，不讲解决方案，不承诺无偿服务，也不打断主结果体验。

## 3. 当前 CTA 边界

当前 CTA 引导用户先关注公众号，再通过回复【重估】获取企业微信添加方式。

页面不直接展示真实企业微信号，不新增二维码图片，不暴露私人微信。

关注公众号或添加企业微信，不等于进入一对一服务。是否进一步沟通，以后续实际问题为准。
