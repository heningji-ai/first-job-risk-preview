### 1. 产品目标

产品不是职业推荐系统。

产品判断的是：

用户自己选择的目标公司类型和目标岗位类型，是否适合现在的他。

系统根据：

- 基础背景。
- 职场性格底色。
- 求职动机。
- 职场通用场景反应。
- 目标岗位场景反应。

生成分屏式目标适配报告。

### 2. 产品定位

首页主张：

你想去的公司，想做的岗位，真的适合你吗？

产品不是：

- MBTI。
- 娱乐测试。
- 职业算命。
- 职业推荐器。
- 性格标签测试。

产品是：

面向大学生 / 应届生第一次求职迷茫的目标适配测试。

基于猎头季哥 21 年招聘经验，帮助用户判断：

- 这个目标能不能进场。
- 这个目标短期能不能适应。
- 这个目标长期会不会消耗。
- 这个目标是该继续冲，还是该调整策略。

### 10. 题库数据结构

每个选项必须支持：

- `mainScore: 0-5`
- `companyScores`
- `roleScores`
- `motivationTags`
- `riskTags`

默认规则：

- `companyScores` 未写入的公司类型，默认 `2.5` 分。
- `roleScores` 未写入的岗位类型，默认 `2.5` 分。
- `mainScore` 必须每个选项都有。
- `riskTags` 用于风险提示，不直接覆盖总分。

### 11. 评分算法

V1.3 必须使用 Goal Fit 新算法。

禁止复用 V1.2 五维 pathFit 加权模型。

核心分数：

- `companyEntryScore`
- `roleEntryScore`
- `companyPersonalityScore`
- `companyBehaviorScore`
- `companyFitScore`
- `rolePersonalityScore`
- `roleBehaviorScore`
- `roleFitScore`
- `motivationFitScore`
- `pairScore`
- `overallScore`

目标公司适配度：

```txt
companyFitScore =
companyEntryScore * 25%
+ companyPersonalityScore * 30%
+ companyBehaviorScore * 30%
+ motivationFitScore * 15%
```

目标岗位适配度：

```txt
roleFitScore =
roleEntryScore * 25%
+ rolePersonalityScore * 25%
+ roleBehaviorScore * 35%
+ motivationFitScore * 15%
```

综合适配度：

```txt
overallScore =
companyFitScore * 45%
+ roleFitScore * 45%
+ pairScore * 10%
```

### 13. 四象限判断

公司四象限比较：

- `companyPersonalityScore`
- `companyBehaviorScore`

岗位四象限比较：

- `rolePersonalityScore`
- `roleBehaviorScore`

四类结论：

1. 高度匹配。
2. 底色适合，但行为 / 方法还不成熟。
3. 短期能适应，但长期高消耗。
4. 不建议作为第一优先方向。

`65-74` 为有条件适配区间。

### 14. 风险规则

风险规则用于结果页文案，不强行覆盖总分。

必须支持：

- 稳定动机 × 创业公司。
- 大厂光环 × 压力耐受低。
- 内向低沟通能量 × 销售 / 市场外联。
- 想有人带 × 创业公司 / 民企。
- 成就创造高 × 螺丝钉环境。
- 边界需求强 × 人情型组织。
- 背景准入弱但行为适配高。
- 准入强但行为适配低。

每次结果至少输出 1 条风险提示。

### 15. 结果页结构

结果页采用分屏卡片，不做一屏长报告。

建议 7 屏：

1. 总结结果
   - 目标公司。
   - 目标岗位。
   - 综合适配度。
   - 一句话结论。

2. 目标公司适配
   - 公司适配度。
   - 准入分。
   - 性格适配。
   - 行为适配。
   - 公司四象限结论。

3. 目标岗位适配
   - 岗位适配度。
   - 准入分。
   - 性格适配。
   - 行为适配。
   - 岗位四象限结论。

4. 最大风险场景
   - 来自 `riskTags` / quadrant / `pairScore`。
   - 写成人话，不写成技术解释。

5. 猎头季哥怎么看
   - 一句专业判断。
   - 像猎头看完候选人后的判断。
   - 不像测评报告模板。

6. 接下来怎么做
   - 最多 3 条行动建议。
   - 必须具体。

7. 公众号引流
   - 关注公众号：猎头季哥人才重估实验室。
   - 二维码路径：`/public/qrcode-wechat.jpg`。
   - 如果二维码不存在，显示占位提示。
