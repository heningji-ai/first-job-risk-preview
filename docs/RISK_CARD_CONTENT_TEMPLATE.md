# 风险卡内容填写模板

本文件是产品方填写模板。请勿把模板占位当作正式风险卡内容。

填写说明：

- `cardId` 可先沿用 H1-H16。
- 每张卡都必须至少配置一个 primaryRiskSignal。
- `flag` 只能作为辅助信号。
- `strongMatch` 只做准入，不加分。
- `protectRules` 任意命中即跳过该卡。
- 本文件不提供正式风险判断，不提供正式建议。
