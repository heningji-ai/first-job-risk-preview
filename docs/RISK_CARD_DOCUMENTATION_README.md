# H1-H16 风险卡文档说明

本次整理形成两份文档：

1. `RISK_CARD_INTERNAL_SPEC_DRAFT.md`  
   内部判断规格表。用于产品方、开发方、测试方确认风险卡触发边界、保护边界、strongMatch、主风险信号和测试用例。此文档不展示给用户。

2. `RISK_CARD_USER_DISPLAY_COPY_DRAFT.md`  
   用户展示内容表。用于结果页、分享页和行动建议模块。只包含用户可理解、可行动的内容，不展示工程字段和触发逻辑。

当前版本仍为产品定义第一版，不应直接视为正式上线配置。正式落配置前还需要完成：

- 统一 cardId 命名；
- 将内部规格转换为 `risk_cards.json` 条件规则；
- 将用户展示内容转换为结果页 copy 配置；
- 补全正式评分分值和权重；
- 更新 test_cases，验证触发与保护边界；
- 确认 H0 fallback 文案；
- 确认是否接入 B1 / MBTI 路径。
