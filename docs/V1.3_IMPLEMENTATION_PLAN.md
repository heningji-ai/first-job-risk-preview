- `docs/V1.2_FREEZE.md`
- `docs/V1.3_GOAL_FIT_REBUILD_SPEC.md`

- `docs/V1.3_MISSING_REQUIREMENTS.md`

### 禁止修改文件

- `src/`
- `public/`
- `package.json`
- `package-lock.json`
- `vite.config.ts`
- `tsconfig.json`
- `README.md`
- 任何现有页面、路由、题库、评分引擎和结果构建器。

### 验收命令

- `git status --short`
- `git diff --name-only`

- 只新增或修改 docs 文件。
- 没有业务代码改动。

- 没有 commit。
